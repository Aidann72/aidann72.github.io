# Aidan Nagle

Personal portfolio website for **Supply Chain Systems & Analytics**, hosted with GitHub Pages.

## Current scope

The site is intentionally a minimal placeholder. Only the homepage introduction is public. The HTML already contains hidden placeholders for future About, Experience, Projects, and Contact sections.

## Repository structure

- `index.html`: homepage and future section placeholders
- `styles.css`: responsive design and accessibility styling
- `.nojekyll`: tells GitHub Pages to publish the static files directly

Project case studies should live in separate public repositories and be linked from the Projects section when they are ready. Do not add confidential employer data, private contact details, credentials, or proprietary files.

## Publishing

GitHub Pages should publish from the `main` branch and the repository root. Updates to the branch are deployed automatically.
